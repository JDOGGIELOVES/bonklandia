# Bonklandia

Bonklandia is a turn-based combat and casino game for the Bonk Fam bloodlines — fight through Degen Valley, spin at the Bonklandia Casino, and cash out chips at the Bonklandia Cashier.

**Production:** [bonklandia.com](https://bonklandia.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Treasury

The game shares a Solana treasury with Bonk Miner and GrokSight. Sync locally:

```bash
npm run treasury:sync
npm run treasury:check
```

## Build

```bash
npm run build
npm start
```
